Anker Solix Display

ESP32-C3 + GC9A01A Round Display Flasher

● Version 1.0.2

📋 Features

  • ☀️ Solar-Leistung (PV)
  • 🔋 Batterie SOC % (Anker Solix E2700 Pro)
  • 🔌 Netzbezug / Einspeisung
  • ⚙️ Config-Portal via WLAN-Hotspot
  • 🕐 Automatische NTP-Zeitanzeige

🚀 Anleitung

  1. ESP32-C3 per USB-C anschließen
  2. Auf Flash klicken und USB-Port auswählen
  3. Nach dem Flash: WLAN "Anker-Display-Setup" verbinden
  4. Browser öffnet sich → WLAN + Anker-Zugangsdaten eingeben
⚠️ Bitte Chrome oder Edge verwenden ⚠️ Seite muss über HTTPS geöffnet werden

⚠️ Funktioniert nur mit Chrome oder Edge
(kein Firefox / Safari – kein WebSerial Support)

Nach dem Flash verbindet sich der ESP als WLAN-Hotspot.
Zugangsdaten werden sicher im Flash gespeichert.
⚠️ Hinweis
Dies ist ein Bastelprojekt und funktioniert solange Anker die inoffizielle Cloud-API zur Verfügung stellt. Eine Garantie gibt es nicht – Anker kann die API jederzeit ändern oder abschalten.

Der Materialeinsatz von ca. 20 € (ESP32-C3 Board + rundes Display) ist für ein Hobbyprojekt dieser Art absolut vertretbar. 🔧
